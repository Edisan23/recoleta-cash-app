# **App Name**: Recoleta Cash App

## Core Features:

- Title Display: Display the title 'Recoleta Cash' prominently.
- Data Table: Show data in a tabular format, including columns for 'Nombre' (name) and 'Monto del Dinero' (amount of money).
- Entry Boxes: Each entry will be displayed in a separate 'cajoncito' (little box).
- Preset Amounts: Offer preset 'bandejitas' (trays) with common amounts: $10,000, $20,000, $30,000, and $40,000 for quick selection. This is implemented as buttons.
- Download Screenshot: Provide an option to download a screenshot of the current screen, which can then be shared.
- Multiple Images: Automatically determine whether one or more images need to be created based on the length of the list.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to symbolize prosperity and trust.
- Background color: Light purple (#EDE7F6), a very pale tint of the primary color to give good contrast.
- Accent color: Pink (#E91E63), analogous to purple, for highlighting and interactive elements. This color will bring attention to details on the screen.
- Font pairing: 'Playfair' (serif) for titles and 'PT Sans' (sans-serif) for body text, combining elegance with readability.
- Use a grid layout to arrange the data table and preset amount options in an organized and visually appealing manner.
- Employ simple and clear icons for the download and share options.
